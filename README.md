# growi-plugin-reading-progressbar

GROWI のページ閲覧時に、スクロール進捗をヘッダー直下にプログレスバーで表示する Script プラグインです。

## 機能

- スクロール量に応じて 0〜100% でバーが伸縮する
- バーの色は GROWI 管理者が設定した primary カラーに自動追従する
- SPA ページ遷移時に進捗を自動リセットする
- 閲覧モードのみ表示（編集モードでは非表示）
- 管理画面（`/admin` 以下）では非表示
- 印刷時は非表示
- プラグイン無効化時にすべてのリスナー・DOM を完全クリーンアップ

## インストール

GROWI の管理画面 `/admin/plugins` でこのリポジトリの URL を入力してインストールしてください。

インストール後、プラグインを有効化するとプログレスバーが表示されます。

> **注意**: コード更新を反映する際は、管理画面でプラグインを**削除 → 再インストール**してください。有効/無効のトグルのみでは zip が取り直されません。

## 開発・ビルド

### セットアップ

```bash
pnpm install
pnpm approve-builds --all   # 初回のみ（pnpm 11+ の場合）
pnpm install                 # esbuild の postinstall を実行するため再度
```

### ビルド

```bash
pnpm build
```

`dist/manifest.json` と `dist/assets/` が生成されます。`dist/` はコミット必須です（GROWI はビルドを実行しないため）。

- [ ] `pnpm build` が成功し `dist/manifest.json` が出力される
- [ ] GROWI で削除 → 再インストール後、DevTools Network で `client-entry-*.js` が 200 で取得される
- [ ] 長文ページでスクロールするとバーが 0% → 100% へ変化する
- [ ] バーの色が GROWI の primary カラーと一致する
- [ ] 別ページに遷移すると進捗が 0% にリセットされる
- [ ] 編集モード中はバーが非表示になる
- [ ] 編集モード終了後にバーが再表示される
- [ ] 管理画面（`/admin`）ではバーが非表示になる
- [ ] 印刷プレビューでバーが表示されない
- [ ] プラグイン無効化でバーが完全に消える
